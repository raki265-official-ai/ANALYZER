# NumPy Analyzer

A Python-based data analysis project that demonstrates the power of the NumPy library for performing fast numerical computations, statistical analysis, and array manipulations.

## 📌 Features

- Create and manipulate NumPy arrays
- Perform mathematical operations
- Calculate statistical measures
- Matrix operations
- Array indexing and slicing
- Data reshaping
- Sorting and filtering
- Performance comparison with Python lists

## 🛠️ Technologies Used

- Python 3.x
- NumPy

1. Clone the repository

```bash

```


Run the project using:

```bash

```

## 📊 Example Operations

- Mean
- Median
- Standard Deviation
- Sum
- Maximum & Minimum
- Matrix Multiplication
- Reshape Arrays
- Random Number Generation

Example:

```python
import numpy as np

data = np.array([10, 20, 30, 40, 50])

print("Mean:", np.mean(data))
print("Sum:", np.sum(data))
print("Standard Deviation:", np.std(data))
```

Output:

```
Mean: 30.0
Sum: 150
Standard Deviation: 14.14
```
